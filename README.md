# Daily Code Commit Guide

## 목적

* 주 5일 알고리즘/코딩 테스트 문제를 풀고 GitHub에 기록한다.
* 꾸준한 학습 루틴을 만들고 코딩 능력 성장을 목표로 한다.

## 운영 방식

### 1. 문제 풀이 빈도

* 주 5일
* 문제 플랫폼은 자유(백준, 프로그래머 등)

### 2. 제출 형식

* 해당 레포에 커밋/push
* 커밋 메시지 예시(미정)

  * `feat: BOJ 1541 - 잃어버린 괄호 풀이`
  * `solve: 프로그래머스 LV1 두 정수의 합`

### 3. 디렉토리 구조 예시
(미정)
```
/problems
  /boj
    1541_잃어버린괄호.java
  /programmers
    lvl1_두정수의합.js
```

### 4. 체크 방식

* 매주 월요일 확인(미정)

## 팀 규칙

* 1회 누락 시 벌점 1점
* 벌점 총 5점 누적 시 커피 돌리기
