# Daily Code Commit Guide

## 목적

* 주 5일 알고리즘/코딩 테스트 문제를 풀고 본인의 Git Repository에 작성
* 꾸준한 학습 루틴을 만들고 코딩 능력 성장을 목표로 한다.

## 운영 방식

### 1. 문제 풀이 빈도

* 주 5일
* 문제 플랫폼은 자유(백준, 프로그래머 등)

### 2. 제출 형식

* 개인 레포에 커밋/push
* 노션에 완료표시

### 3. 체크 방식

* 날마다 노션을 통해 체크

## 팀 규칙

* 1회 누락 시 벌점 1점
* 벌점 총 5점 누적 시 커피 돌리기
